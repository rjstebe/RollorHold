Roll or Hold

A Calculator for Dice Rolling Games

Yahtzee
Yam-Slam
How to Use

Roll or Hold is a calculator that automatically calculates which dice a player should hold and which they should keep to obtain the highest score for the turn in a game of Yahtzee or Yam-Slam. You can input the current dice, rolls remaining, and score options, which all factor into the calculator's suggestion.

The number of rolls remaining in the turn can be given to the calculator in the rolls remaining field. The number of rolls remaining in the turn changes the probability of ending the turn with different dice values, and so changing this value affects the calculation.

Pressing the buttons above or below a die changes the die's value. This is how you select the die values that the calculator will use to make a recommendation.

Beneath each die, a yellow rectangle show's the calculator's recommendation for that die. The recommendation will be either to "Roll" the die or to "Hold" the die.

The "score options" button reveals all the scoring options for the current game. This button can be pressed again to close the scores panel.

In Yahtzee, each score option shows the calculated score that would be gained by the player if the current set of dice values were to be entered into that option. The checkbox determines whether or not the scoring option has already been used. Checked checkboxes represent a scoring option that has already been used, and an unchecked box represents an option that is yet to be used.

In Yam-Slam, each score option shows the score that the option would provide, which is unchanged by the dice values. The number next to each option represents the number of chips of that option remaining, and can be altered to match the state of the game.

The symbol shows a tooltip about the feature beside it when hovered over, or when tapped on a mobile device.
